Master-Detail

This is a best practice implementation of the master detail design pattern in UI5.
Choose one of the access points below to launch the app.

Standalone

These files can be used to run the application as a standalone application in the SAP Web IDE or after building and serving with bower/grunt (expects a UI5 installation available at the server root, so this will not work without this build and the grunt-based server).
  • testService.html - start the app with mock data
  • build.html - start the app with code built for production with mock data
  • test/unit/unitTests.qunit.html - run all unit tests
  • test/integration/opaTests.qunit.html - run all integration tests
